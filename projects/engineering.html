Engineering Principles: Digital & Analog Circuits, PCB Design, and LiDAR Development

Published: 2024-05-16
Category: Engineering Principles
Author: Zhipeng Zhou

This article details my project experience and technical achievements in digital circuits, analog circuits, PCB design and manufacturing, and LiDAR embedded development. Through these projects, I systematically mastered the entire process from circuit principles to practical hardware implementation, laying a solid foundation for subsequent embedded system development and hardware design.

I. Digital Circuit and Analog Circuit Design

In digital circuit and analog circuit courses, I completed multiple experimental projects covering basic circuit principles and advanced circuit design techniques:

1.1 Digital Circuit Projects

For digital circuit design, I implemented various logic functions and systems using hardware description languages (VHDL/Verilog) and realized them on FPGA platforms:

  • Designed and implemented a 4-bit ALU (Arithmetic Logic Unit)
  • Created a digital clock with multiple display modes
  • Developed a traffic light control system with state machine implementation
  • Built a simple CPU that could execute basic instructions
// Verilog Code Sample - 4-bit ALU
module alu_4bit(
    input [3:0] A,
    input [3:0] B,
    input [1:0] ALU_Sel,
    output [3:0] ALU_Out,
    output CarryOut
);
    reg [4:0] ALU_Result;
    assign ALU_Out = ALU_Result[3:0];
    assign CarryOut = ALU_Result[4];
    
    always @(*)
    begin
        case(ALU_Sel)
            2'b00: ALU_Result = A + B;     // Addition
            2'b01: ALU_Result = A - B;     // Subtraction
            2'b10: ALU_Result = A & B;     // Logical AND
            2'b11: ALU_Result = A | B;     // Logical OR
        endcase
    end
endmodule

1.2 Analog Circuit Projects

In analog circuit design, I focused on signal conditioning, amplification, and filtering applications:

  • Designed and built operational amplifier circuits (inverting, non-inverting)
  • Created active filters (low-pass, high-pass, band-pass) for signal processing
  • Implemented a precision rectifier circuit for signal conversion
  • Developed a temperature-controlled circuit using thermistors
  • Built an audio amplifier circuit with tone control

II. PCB Design and Manufacturing

After mastering circuit principles, I advanced to PCB design and manufacturing, learning industry-standard EDA tools and techniques:

2.1 PCB Design Software Proficiency

I gained proficiency in professional PCB design software:

  • Altium Designer: Schematic capture, component placement, routing
  • KiCad: Open-source alternative for PCB design
  • Eagle: PCB layout and design verification

2.2 PCB Manufacturing Process

I completed the entire PCB manufacturing process, from design to finished product:

  1. Schematic design: Creating circuit diagrams with proper electrical connections
  2. Component selection: Choosing appropriate components with consideration for power, size, and cost
  3. PCB layout: Arranging components and routing traces for optimal performance
  4. Design rule checking (DRC): Verifying design against manufacturing constraints
  5. Gerber file generation: Preparing fabrication files for manufacturing
  6. PCB fabrication: Working with manufacturing services to produce the physical board
  7. Component soldering: Manually soldering components to the manufactured PCB
  8. Testing and debugging: Verifying functionality and troubleshooting issues

III. LiDAR Embedded System Development

Building on my foundation in circuits and PCB design, I explored embedded systems development with a focus on LiDAR technology:

3.1 LiDAR Principles and Applications

I studied the fundamental principles of LiDAR (Light Detection and Ranging) technology:

  • Point cloud generation and processing
  • Sensor fusion with other sensing modalities

3.2 Embedded System Integration

I integrated LiDAR sensors with embedded systems using microcontrollers:

  • Interfaced with LiDAR sensors (Serial, I2C, SPI communication)
  • Implemented data acquisition and processing algorithms
  • Optimized code for real-time performance on resource-constrained devices
  • Developed visualization tools for LiDAR data
  • Created a complete obstacle detection and avoidance system
