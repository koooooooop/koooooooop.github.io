Overview

Vision-Guided Robotics Research

This project documents my exploration in the fields of robotics, embedded systems, and machine vision. Through practical applications and theoretical study, I've gained comprehensive knowledge in robotics fundamentals that bridges hardware design with software implementation.

My research focuses on the integration of computer vision and machine learning techniques with robotic systems to enable perception and autonomous interaction with real-world environments.

Key Research Areas

  • Computer Vision & Perception
  • Embedded Control Systems
  • Sensor Fusion & Integration
  • Autonomous Navigation
  • Robot Operating System (ROS)

Development of a wheeled mobile robot platform capable of autonomous navigation and obstacle avoidance using computer vision and sensor fusion.

Design and construction of a 5-DOF robotic arm with computer vision integration for autonomous pick-and-place operations in dynamic environments.

  • Camera integration for object detection
  • Custom gripper design for various object shapes
  • User interface for both autonomous and manual control
