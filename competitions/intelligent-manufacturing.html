Competition Overview

I participated in the First Chongqing University Intelligent Manufacturing Competition, which focused on Siemens PLC (Programmable Logic Controller) programming and automation solutions. This competition provided a valuable opportunity to apply theoretical knowledge in practical industrial automation scenarios.

The competition challenged participants to design, program, and implement automation solutions for simulated manufacturing processes using Siemens PLC hardware and software. Our team demonstrated innovative thinking and strong teamwork throughout the competition phases.

PLC Programming

Developed ladder logic programs for Siemens S7-1200/1500 PLCs to control simulated manufacturing processes

HMI Design

Created user-friendly Human-Machine Interface designs using SIMATIC WinCC for process monitoring and control

System Integration

Implemented integrated automation solutions connecting sensors, actuators, and control systems

Technologies Used

Siemens Automation Technology

  • SIMATIC S7-1200/S7-1500 PLCs - For main control system programming
  • TIA Portal - Integrated development environment for PLC programming and HMI design
  • SIMATIC WinCC - For human-machine interface development
  • SIMATIC STEP 7 - PLC programming software
  • PROFINET/PROFIBUS - Industrial network protocols for device communication

Programming Languages

  • Ladder Logic (LAD) - Primary programming language for control logic
  • Function Block Diagram (FBD) - For complex function implementation
  • Structured Text (SCL) - Text-based programming for complex algorithms
  • Statement List (STL) - Low-level language for optimization

Project Development

Competition Registration and Preparation
2 weeks before the competition

Our team registered for the competition and began preparing by studying Siemens PLC programming fundamentals, reviewing automation concepts, and practicing with TIA Portal software. We also analyzed previous competition challenges to understand the expected requirements.

Preliminary Round
First day of the competition

In the preliminary round, we were tasked with implementing basic automation logic for a simulated production line. Our team successfully programmed the PLC to control sequential operations, handle error conditions, and manage process timing, qualifying us for the next round.

Main Competition Challenge
Second day of the competition

The main challenge involved designing and implementing a complete automation solution for a complex manufacturing process. We developed a comprehensive control system with the following features:

  • Multi-state machine control with automatic and manual modes
  • Safety interlocks and emergency stop functionality
  • Process visualization through a custom HMI interface
  • Data logging and parameter adjustment capabilities
  • Alarm management and system diagnostics
Final Presentation and Demonstration
Final day of the competition

We presented our solution to the judges, explaining our design choices, implementation approach, and demonstrating the working system. The judges evaluated our solution based on functionality, innovation, reliability, and presentation quality.

Technical Challenges and Solutions

Challenge 1: State Machine Implementation

Problem: Creating a reliable state machine to manage multiple operation modes and transitions between them.

Solution: Implemented a structured state machine using Function Block Diagram (FBD) programming, with clear state definitions, transition conditions, and fault handling logic.

Challenge 2: Communication Protocols

Problem: Establishing stable communication between multiple devices in the automation network.

Solution: Used PROFINET protocol with properly configured addressing and timing parameters, implementing error detection and recovery mechanisms.

Challenge 3: User Interface Design

Problem: Creating an intuitive operator interface that provides clear process information and control options.

Solution: Designed a hierarchical HMI structure with main overview, detailed process views, parameter screens, and alarm displays, following ergonomic design principles.

Challenge 4: Real-time Process Optimization

Problem: Optimizing system performance to meet timing and efficiency requirements.

Solution: Analyzed program execution time, implemented optimized code structures, and used high-performance instructions to improve system responsiveness.

Reflection and Learning

Participating in the Intelligent Manufacturing Competition was an invaluable learning experience that bridged theoretical knowledge with practical application. The competition taught me several important lessons:

1. Industry-Ready Skills

Working with industry-standard Siemens PLC equipment provided direct exposure to technologies used in real manufacturing environments. These hands-on skills are directly transferable to industrial automation careers.

2. Systematic Problem Solving

The competition challenges required systematic approaches to problem decomposition, solution design, and implementation. I learned to tackle complex automation problems by breaking them down into manageable components.

When facing unexpected issues during the competition, our team had to quickly adapt our approach and find alternative solutions. This experience improved my ability to think on my feet and make effective decisions under pressure.

4. Interdisciplinary Integration

The competition demonstrated how different engineering disciplines converge in industrial automation, requiring knowledge of electronics, programming, mechanical systems, and human factors.
